# Animación de Círculos con Canvas

## Descripción
Este proyecto utiliza la **API Canvas de JavaScript** y **Programación Orientada a Objetos** para crear una animación 2D.  
Se dibujan círculos que se mueven dentro de la ventana del navegador y rebotan al tocar los bordes.

---

## Tecnologías
- JavaScript  
- HTML5 Canvas  
- API Canvas 2D  

---

## Funcionamiento
- El canvas se ajusta automáticamente al tamaño de la ventana.
- Se define una clase `Circle` para representar cada círculo.
- Cada objeto tiene posición, radio, color, texto y velocidad.
- Los círculos se mueven y cambian de dirección al colisionar con los límites.
- La animación se ejecuta usando `requestAnimationFrame()`.

---

## Ejecución
Abre el archivo HTML en un navegador web compatible con Canvas.
